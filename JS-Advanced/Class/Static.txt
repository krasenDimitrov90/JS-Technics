static members
Можем да си съсдаваме статични методи и статични пропъртита

Разликата между методите в класа които се обвързват с обекта и като искаме да ги извикаме 
трябва да напишем името на обекта точка името на метода
Пример 
class Person {
    constructor (firstName, lastName, age, email) {
        this.firstName = firstName,
    }

    toString() {
        return `${this.firstName}`;
    }
}

let person = new Person('Anna', 'Simpson', 22, 'anna@yahoo.com');
console.log(person.toString());

Разликата е че когато искaме да извикаме статичен метод или пропърти е че вместо името на обекта 
пишем името на класа точка името на статичния метод или пропърти


Добър пример за статични мембъри е Math библиотеката на JS

Math.PI - Math е класа , а PI e property
Math.pow() - Math e класа , а pow() e метод

Т.е. можем да си създадем подобна библиотека с наши си методи които да ползваме
