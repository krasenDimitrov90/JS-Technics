geter-a е функция която връща резултат
всеки път когато е извикана тя изчислява резултата на база на текущите стойности
които са и зададени (най-често от конструктора)
Т.е. ако в конструктора имаме например 'x' който сме задали да е равен на 2 ето така this.x = 2
а geter функцията връща резултата от x*2 всеки път когато я извикаме тя първо ще изчисли 
и тогава ще върне резултат, т.е. ако по надолу в кода променим 'х' да е равно на 4 например 
и извикаме гетъра тя ще изчисли на база на новата стойност и ще върне 8

class Circle {
    constructor(radius) {
        this.radius = radius;
    }
    get diameter() {
        return this.radius * 2;
    }
}

let c = new Circle(2);
console.log(c.diameter); Output 4
c.radius = 4;
console.log(c.diameter);Output 8
