.sort() 
	Сортира масива който му е зададен.НЕ СЕ СЪЗДАВА НОВ МАСИВ
	Пример:
	let arr = [5,3,4,8,7,9,]
	let sortedArr = arr.sort((element1 , element2) => {    - element1 и element2 можем да кръстим както искаме
		return element1 - element2         - знакът '-' не е минус , а ТИРЕ.

	})
	sortedArr = [3,4,5,7,8,9]
	arr = [3,4,5,7,8,9] т.е. и двата масива сочат към една и съща памет в компа
				ако променим по надолу който и да е от двата , другия също ще приеме неговата стойност
	в този случай сортираме от малко към голямо число
	ако напишем 
	return element2 - element1 ще сортираме от голямо към малко
сортиране по дължина на стринг
	
	let arr = ['b','ccc','aa',]
	let sortedArr = arr.sort((a , b) => {
		return a.length - b.length;
	});
	sortedArr = ['b','aa','ccc']
азбучно сортиране на стринг
	return a.localeCompare(b);
	sortedArr = ['aa','b','ccc']

Сортиране по няколко параметъра
	в случая по дължина и азбучно
	let arr = ['bbb','ccc','aaa',]
	let sortedArr = arr.sort((a , b) => {
		return a.length - b.length || a.localeCompare(b);
	});
	sortedArr = ['aaa','bbb','ccc']
	В този пример функцията първо сравнява по дължина , след което азбучно
	Ако не бяхме направили проверка a.localeCompare(b) нямаше да се промени нищо , т.е = ['bbb','ccc','aaa',]
