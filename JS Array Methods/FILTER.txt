.filter() 
	Филтрира масив от елементи и връща тези който отговарят на подадено условие
Пример let arr = [1,2,3,4,5]
	let arrFiltered = arr.filter((el) => {
		if (el % 2 === 0 )
			return el;
		})
	новият масив е [2,4] защото само те са четни и отговарят на условието което сме задали
Тази функциця също връща нов масив , който се заделя в отделна памет на компютъра
