.map()
   Тази функция връща нов масив (т.е. в отделна памет), като минава през всеки един елемент на масива и ако сме му задали 
	може да изпълнява например математически операции 
Пример
	let arr = [1,2,3]
	let arrMaped = arr.map((el) => el*2) - резултатът е нов масив със стойностите умножени по 2 -[2,4,6]
Пример за парсване на стрингови стйности в числа
	let arr = ['1','2','3']
	let arrAsNumbers = arr.map(Number) rezultat [1,2,3]
.map() Не премахва елементи т.е. ако искаме да премахваме елементи позваме метода .filter()
